# lerp.js [![npm version](https://badge.fury.io/js/lerpjs.svg)](https://badge.fury.io/js/lerpjs)

The method of linear interpolation (lerp) to easing animations.

## Example

```javascript
import lerp from 'lerpjs';

lerp(0, 100, 0.2); // => 20
```
